# Observability
## Observability signals/pillars (관측 데이터 유형/수단)
- Metrics (메트릭): 수치화된 지표 (CPU 사용률, 요청 수, 응답 시간 등)
- Logs (로깅): 이벤트 기록 (에러 메시지, 상태 변화, 사용자 액션 등)
- Tracing (트레이싱): 분산 시스템에서 요청 흐름 추적 (마이크로서비스 간 호출 관계, 지연 구간 파악 등)
- Profiling (프로파일링): 코드 실행 단위에서 성능 분석 (CPU/메모리 사용 패턴, 함수 호출 빈도 등)
